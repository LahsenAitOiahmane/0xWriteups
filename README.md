# 0xWriteups

A curated collection of CTF writeups, challenge notes, and solving resources by **l27sen**.

## Repository Overview

This repository groups writeups by category and challenge source, including:

- **PWN** (binary exploitation)
- **REV** (reverse engineering)
- **WEB** (web exploitation)
- **CRYPTO** (cryptography)
- **MISC** (miscellaneous challenge writeups and supporting files)
- **VM** (this folder contains mobile/VM-oriented challenge writeups and artifacts)
- **REV-PWN-RootMe** (RootMe reverse/pwn challenge notes)

## Structure

```text
0xWriteups/
├── CRYPTO/
├── MISC/
├── PWN/
├── REV/
├── REV-PWN-RootMe/
├── VM/
└── WEB/
```

Each challenge folder usually contains one or more of the following:

- A writeup file (`README.md`, `WRITEUP.md`, or `*_writeup.md`)
- Analysis notes and command outputs (`outputs.txt`)
- Solver or exploit scripts (`solve.py`, `exploit.py`, etc.)
- Challenge binaries/files used during analysis

## Notes

- Content is for **educational and legal CTF practice**.
- Some folders may include local test artifacts from challenge environments.
